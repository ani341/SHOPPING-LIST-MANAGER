# 🛒 Shopping List Manager

A beginner-friendly Python Command Line Interface (CLI) application that creates, sorts, and manages a user's grocery list using core Python data structures and string manipulation techniques.

---

## ✨ Features

* **Personalized Greeting:** Welcomes the user with automatically title-cased names and customized initials upon exit.
* **Dynamic List Creation:** Collects grocery items from user input and automatically adds default staples (like `eggs`).
* **List Sorting:** Automatically sorts the shopping list alphabetically.
* **Budget Management:** Uses list popping to trim down the grocery list based on how many items the user can afford.
* **Data Immutability:** Converts the finalized shopping list into an immutable Python tuple for safe storage.

---

## 🛠️ Concepts Demonstrated

This mini-project highlights core Python concepts learned in early programming practice:
* **Data Structures:** Lists (`append()`, `sort()`, `pop()`, `len()`) and Tuples.
* **String Manipulation:** `.title()`, `.capitalize()`, `.split()`, `.upper()`, and string indexing.
* **User Interaction:** Interactive terminal prompts using `input()` and data type casting (`str()`, `int()`).
